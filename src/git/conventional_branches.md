# Conventional Branches

## Template

```bash
<type>/<jira-id>
```

## Types

| **Type** | **Description**                                    |
|----------|----------------------------------------------------|
| build    | build system and dependency changes                |
| chore    | non-code tasks like dependency, docs, updates etc. |
| feat     |                                                    |
| fix      |                                                    |
| hotfix   |                                                    |

## Find Branches

```bash
git branch --list "fix/*"
```
